The code used for the reported analyses is archived with the tagged source release at Zenodo \url{https://doi.org/10.5281/zenodo.20351628} and mirrored in the public repository \url{https://github.com/KleinChen42/EmbodiedStressBench.git} at release tag \texttt{v0.1.0-scirep}, which resolves to the archived release commit. The repository contains the experiment runners, stressor definitions, RGB-D target-source baselines, detector bridge interfaces, reporting scripts, table-generation scripts, external RGB-D validation scripts, and manuscript figure scripts used in this study. Reproduction commands and artifact checks are documented in the reproducibility README, the Scientific Reports claim-evidence audit, and the release-package metadata.
