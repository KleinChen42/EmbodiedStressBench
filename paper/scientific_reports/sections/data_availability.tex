The processed data supporting the tables and figures in this manuscript are organized in the public project repository at \url{https://github.com/KleinChen42/EmbodiedStressBench.git} and in the archived release associated with this manuscript. The repository release contains the generated CSV source tables, qualitative case manifest, experiment configurations, JSON schema, result-to-claim audit, figure source data, and scripts used to regenerate the Scientific Reports figures and tables. The source-data package is organized under \texttt{scientific\_reports\_revision\_20260521/release\_package}. The full per-episode JSON outputs are larger than the processed source-data package and are not required to regenerate the manuscript tables and figures; the processed source data, schema, configs, and reduced reproducibility subset are the data supporting the reported results.
