The processed data supporting the tables and figures in this manuscript are archived with the tagged source release at Zenodo \url{https://doi.org/10.5281/zenodo.20351628} and mirrored in the public repository \url{https://github.com/KleinChen42/EmbodiedStressBench.git} at release tag \texttt{v0.1.0-scirep}, which resolves to the archived release commit. The source-data release contains generated CSV source tables, figure source data, the qualitative case manifest, experiment configurations, JSON schema, result-to-claim audit, checksum manifest, and scripts used to regenerate the Scientific Reports figures and tables. For the external YCB-V/BOP RGB-D validation, the release includes detector/source summaries, object-level summaries, scene-level summaries, and threshold-sensitivity source data for the default, 0.10/0.10, and 0.30/0.25 detector-threshold settings. The underlying YCB-V/BOP RGB-D frames are public third-party data and are not redistributed; this study uses their ground-truth visible masks, boxes, camera intrinsics, and object names as external target-generation references. The full per-episode JSON outputs from the large simulation sweeps and the full per-record YCB-V indices are larger than the processed source-data package and are not required to regenerate the manuscript tables and figures; the processed source data, schemas, configs, and reduced reproducibility subset are the data supporting the reported manuscript results.
