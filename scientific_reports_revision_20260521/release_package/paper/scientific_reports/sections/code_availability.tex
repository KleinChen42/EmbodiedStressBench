The code is available from \url{https://github.com/KleinChen42/EmbodiedStressBench.git}. The repository contains the experiment runners, stressor definitions, RGB-D target-source baselines, detector bridge interfaces, reporting scripts, table-generation scripts, external RGB-D validation scripts, and manuscript figure scripts used in this study. The submitted version will be identified by the release tag \texttt{v0.1.0-scirep}, archived with a Zenodo DOI before journal submission. Reproduction commands and artifact checks are documented in the reproducibility README, the Scientific Reports result-to-claim audit, and the release-package metadata.
