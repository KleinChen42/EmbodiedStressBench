The processed data supporting the tables and figures in this manuscript are organized in the public project repository at \url{https://github.com/KleinChen42/EmbodiedStressBench.git}. The source-data release contains the generated CSV source tables, qualitative case manifest, experiment configurations, JSON schema, result-to-claim audit, figure source data, queue-completion logs, and scripts used to regenerate the Scientific Reports figures and tables. A Zenodo DOI for the archived GitHub release will be inserted before journal submission. For the external YCB-V/BOP RGB-D validation, the release includes detector/source summaries, object-level summaries, scene-level summaries, threshold-sensitivity source data, and queue logs for the default, 0.10/0.10, and 0.30/0.25 detector-threshold settings. The underlying YCB-V/BOP RGB-D frames are public third-party data; this study uses their ground-truth visible masks, boxes, camera intrinsics, and object names as external target-generation references. The full per-episode JSON outputs from the large simulation sweeps and the full per-record YCB-V indices are larger than the processed source-data package and are not required to regenerate the manuscript tables and figures; the processed source data, schemas, configs, and reduced reproducibility subset are the data supporting the reported manuscript results.
